# ✨ Glowing Light Project

A simple front-end project built using **HTML** and **CSS** that demonstrates the use of CSS Transitions, Transforms, and Glow Effects to create an interactive and visually appealing UI element.

## 🚀 Features

- Smooth hover animations
- CSS Transform effects (scale, rotate, etc.)
- CSS Transition effects for seamless interaction
- Modern glowing light design
- Beginner-friendly project structure
- Responsive and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📚 Concepts Practiced

This project helped me understand:

- CSS `transition`
- CSS `transform`
- Hover effects
- Box-shadow glow effects
- Basic UI animation techniques
- Modern front-end styling

## 🎯 Project Goal

The goal of this project was to strengthen my understanding of CSS animations and interactive design by creating a glowing light effect using only HTML and CSS.

🌱 What I Learned
While building this project, I learned how transitions and transforms work together to create smooth animations and improve user experience. I also explored glow effects using CSS shadows and visual styling techniques.
